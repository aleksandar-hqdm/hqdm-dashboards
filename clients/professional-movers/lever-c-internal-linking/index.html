Schema deploy plan (SCH-01..14)

Every new + upgraded page ships with the right schema markup at publish, not retrofitted. Four schema types in the deployment matrix:

LocalBusiness

All HP-01..05 + new /locations/ pages. Includes USDOT/MCICC/CVED IDs, openingHoursSpecification, areaServed (Detroit metro polygon), aggregateRating from GBP review count.

Service

All SP-01..07 service pages. provider points to LocalBusiness; serviceType + areaServed + termsOfService + offers (price range from quote estimator).

FAQPage

BLG-01..06 commercial-intent blog posts + SP-01..07 service-page FAQ sections. Matches AIO-citation pattern from LLM Visibility mini-report (Q&A structure).
