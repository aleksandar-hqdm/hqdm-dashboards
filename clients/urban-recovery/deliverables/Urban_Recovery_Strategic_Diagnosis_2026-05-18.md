## 06 Root Cause, Ranked

**[PRIMARY · P0]** **Service-page conversion surface is thin → 9,269 local-intent queries rank pos 1–3 at near-zero CTR.** `/services/*` (5 URLs: alcohol-detox / drug-detox / opioid-detox / benzo-detox / short-term-residential) sit on doorway templates with **zero JSON-LD** and no intake-form depth. `recovery center brooklyn` ranks pos 3.55 with **13,770 impressions / 0 clicks in last 180 days**. `drug rehab new york city` ranks 7,100 imp / 1 click. The GBP authority is converting on the GBP-tagged homepage path (`/?utm_source=gmb` = 3,275 clicks / 1.18% CTR) but the organic SERP service-page entries leak 100% of clicks. SP-01 + SP-02 + HP-01 + ON-02 = the structural conversion fix; HP-01 alone is a one-line schema edit (homepage JSON-LD telephone 646-960-6656 → GBP-canonical 646-347-1892 — the only schema block sitewide is currently misaligned).

**[PRIMARY]** **GBP depth gap on the pin — photos 7 / Posts 1 / 1-star share 23%.** Competitor pins gaining the most all have visible photo + Post cadence + review velocity (Ascendant 90 photos, Mountainside 30; Elevate Point +88 cells with 14 reviews and visible Post cadence). Photo count is a documented E-E-A-T proxy + Maps freshness ranker. The 23% 1-star share (26 of 111 reviews) is a documented YMYL Maps deprioritization signal — positive-volume dilution via +20 reviews/month is the only competitive lever a single-location pin can pull at this scale. Compounded by `place_topics: null` (Google has not yet derived enough review-content density to assign topical tags). GBP-01 through GBP-08 directly target this gap (Push 1).

**[SECONDARY]** **Cross-borough Maps non-diffusion — 8/40 cells at 5-coord matrix.** UR doesn't appear once at Downtown Brooklyn / Williamsburg / Crown Heights / Sunset Park / Coney Island. Single Red Hook pin Maps reach hard-capped at the 3.25-mi grid radius. Cross-borough capture is **organic-only** — LP-02 upgrades on existing /locations/* URLs (data-validated: every cross-borough target already has a verbose-slug page with 800-1,700 words, no schema, getting impressions but zero clicks). Highest-leverage single cross-borough URL: `/locations/recovery-center-in-sunset-park-ny` at **1,722 words / 3,152 imp / 26 clicks / pos 19.58** — one push from page 1.

**[OPPORTUNITY]** **SNF category drift on `inpatient rehab` — 85 → 75 cells (−10) driven by Brooklyn Center for Rehabilitation and Nursing entering 0 → 102 cells.** Skilled-nursing facility is not an addiction-treatment competitor but shares the entity-class signal. UR's existing `/services/short-term-residential-program` page doesn't use the word "inpatient" in primary framing. ON-02 page refresh in M2 lands explicit **"Inpatient Drug & Alcohol Rehab Brooklyn — 24-Hour Medical Supervision"** framing; GBP Posts use "Inpatient" phrasing 1× per month; full SP-03 rebuild Q3 after SP-01/SP-02 template proven.
